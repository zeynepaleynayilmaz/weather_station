# weather_station

ESP8266 ve Arduino IDE kullanarak sıcaklık, nem, basınç,
yağmur ve hava kalitesi sensörlerini entegre ettiğim bir hava
durumu istasyonu geliştirdim.

MQTT protokolü üzerinden Adafruit IO’ya veri iletimi sağladım;
verileri anlık izleme ve geçmiş grafiksel analiz imkânı sundum.
